### Downloading repository
```
› git clone https://github.com/Vinnybrunn00/Bot-Telegram-All-Functions.git
```

### Opening folder
```
› cd Bot-Telegram-All-Functions
```

### Installing dependencies
```
› pip install -r requirements.txt
```


### Starting Bot
```
› python3 main.py
```
